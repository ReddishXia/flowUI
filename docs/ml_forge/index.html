Why MLForge

Everything you need.
Nothing you don't.

⬡

Visual Node Graph

Drag in layers, connect nodes. Build entire architectures without touching a line of code.

⟳

Auto Shape Propagation

Connect a Conv layer and in_channels fills itself. After Flatten, Linear's in_features is calculated automatically. No more shape math.

↗

Export to PyTorch

Generate a clean standalone train.py. Run it without any dependency on MLForge.

◎

Live Loss Curves

Watch your model train in real time and run inference afterwards.

⬢

Built-in Datasets

Drop in MNIST, CIFAR-10, and more. Input shapes auto-fill. Validation splits handled for you.

⊕

Robust Error Checking

Shape mismatches get caught before you run. MLForge prevents broken pipelines before they happen.

How it works

Four steps to a
trained model.

  1. 01

    Load your data

    Drop in a dataset node. Chain transforms. End with a DataLoader.

  2. 02

    Build your model

    Connect layers visually. Shapes propagate automatically as you build.

  3. 03

    Configure training

    Wire your model and data to a Loss + Optimizer node. Pick your settings. Hit RUN.

  4. 04

    Export & ship

    Export to a standalone train.py.
